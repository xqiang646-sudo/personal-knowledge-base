从第一性原理理解坐标系转换

一个物理点不会因为换坐标系而移动，改变的是描述它的数字。坐标变换的任务，就是明确“同一个几何对象在另一个基和原点下怎样表示”。

贡献角色：导师原创整理　对象：感知、定位与规划入门学习者　证据状态：通用刚体变换推导

统一约定：列向量、右手坐标系；^A T_B 表示从 B 坐标转换到 A 坐标，也就是“B 在 A 中的位姿”。

对象与记号旋转和平移齐次变换组合与求逆二维例子三维、四元数与时间自检与误区资料

先把“点、向量、坐标系”分开

  • ^A p：同一个几何点在坐标系 A 下的坐标。
  • ^A R_B：把 B 系中的方向坐标旋转到 A 系；三列分别是 B 的三个单位轴在 A 中的表示。
  • ^A t_B：B 系原点在 A 系中的坐标。
  • ^A T_B：把 B 系坐标直接转换到 A 系的刚体变换。
^A p = ^A R_B · ^B p + ^A t_B

点依赖原点，因此要加平移；方向、速度增量和法向量没有原点，不应直接加平移。

二维旋转：新坐标来自基向量投影

采用车辆常见约定：x 向前、y 向左、逆时针为正。B 相对 A 逆时针旋转 θ 时：

           [ cosθ  -sinθ ]
^A R_B =  [ sinθ   cosθ ]

^A p = ^A R_B · ^B p + ^A t_B

旋转矩阵的列是新坐标轴在旧坐标系中的方向。它满足：

RᵀR = I        det(R) = 1        R⁻¹ = Rᵀ

主动还是被动：“旋转一个向量”和“换一个坐标系”常使用互为转置的矩阵。不要靠记忆正负号；先写出 ^A R_B 的含义，再代入单位轴检查。

齐次坐标：把旋转和平移放进一次乘法

          [ ^A R_B   ^A t_B ]
^A T_B = [  0 0 0       1   ]

[ ^A p ]           [ ^B p ]
[   1   ] = ^A T_B [   1   ]

齐次坐标最后一维同时区分对象类型：

点：     [p, 1]  → 旋转 + 平移
方向：   [v, 0]  → 只旋转，不平移

组合与求逆：右边的变换先发生

传感器 S 的点先到车体 B，再到地图 M：

^M T_S = ^M T_B · ^B T_S

^M p = ^M T_B · ^B T_S · ^S p

矩阵一般不可交换，乘法从右向左作用。若已知 ^A T_B=[R,t]，反向变换为：

          [ Rᵀ   -Rᵀt ]
^B T_A = [ 0        1 ]

逆变换中的平移是 -Rᵀt，不是简单的 -t。

二维数值例子：车辆点变到地图

车辆原点在地图 (10,5)，yaw 为 90°；车辆系中有一点 ^B p=(2,1)。

         [ 0  -1 ] [ 2 ]   [ 10 ]   [ 9 ]
^M p =  [ 1   0 ] [ 1 ] + [  5 ] = [ 7 ]

回代：^B p = Rᵀ(^M p - t) = (2,1)

解释：车辆前方 2 米在地图中向上移动 2 米，车辆左侧 1 米在地图中向左移动 1 米，所以结果是 (9,7)。

三维工程里还要处理四元数和时间

旋转顺序

如果使用 yaw/pitch/roll，必须声明是固定轴还是随体轴以及乘法顺序；不同顺序结果不同。

四元数顺序

nuScenes 记录为 w,x,y,z，ROS 消息字段通常排列为 x,y,z,w。读取接口前必须核对并归一化。

静态外参

传感器相对车体的安装关系通常近似固定，可写为 ^E T_S。

动态位姿

车体相对地图随时间变化，应写成 ^G T_E(t)；点和变换必须对应同一时刻或经过插值。

^G T_S(t) = ^G T_E(t) · ^E T_S

源传感器 S₁ 在 t₁ 的点变到目标传感器 S₂ 在 t₂：
^S₂ T_S₁ = [^G T_S₂(t₂)]⁻¹ · ^G T_S₁(t₁)

相机投影还需要内参和有效深度：先用外参把三维点变到相机坐标，确认 z>0，再用相机内参投影到像素；只做外参不能得到正确像素。

四类自检与常见错误

  1. 原点测试：令 ^B p=0，结果应该正好是 ^A t_B。
  2. 单位轴测试：分别变换 B 的 x/y/z 单位轴，检查是否与 A 中预期方向一致。
  3. 往返测试：T⁻¹(Tp) 应在数值容差内回到原点。
  4. 矩阵测试：RᵀR=I、det(R)=1、TT⁻¹=I。

方向写反

没有声明 T_AB 是 A→B 还是 B→A，导致矩阵正确、语义错误。

顺序写反

传感器→车体→地图却把矩阵按口头顺序从左向右作用。

约定混用

行/列向量、左/右手系、度/弧度、欧拉角顺序或四元数分量混在一起。

忽略时间

拿 t₁ 的点配 t₂ 的 ego pose，静态外参看似正确但运动目标和背景都会错位。

错误处理向量

给速度、法向量或方向加了平移；法向量在非刚体变换下还有不同规则。

把经纬度当直角坐标

经纬度需先选择基准、投影和局部原点，不能直接塞进普通 SE(3) 矩阵。
