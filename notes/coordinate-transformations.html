从线性代数到坐标系转换

一个物理点不会因为换坐标系而移动，改变的是描述它的数字。本课先补向量、基和矩阵，再从换基推导旋转、平移、齐次变换与多传感器坐标链。

参考路线：高翔等《视觉 SLAM 十四讲：从理论到实践（第 2 版）》第 3 讲“三维空间刚体运动”，并用第 4 讲解释 SO(3)/SE(3) 与局部增量的入门意义。

原创边界：本文针对自动驾驶数据和传感器链重新组织推导、例题与练习，不复刻书中正文或插图，也不能替代原书。统一使用列向量、右手坐标系；^A T_B 表示 B→A。

线性代数先修：几何对象、基和坐标

几何向量是空间中的方向与长度；坐标列向量是它在某组基下的数字。设正交基矩阵 E=[e₁ e₂ e₃]，则：

a = a₁e₁ + a₂e₂ + a₃e₃ = E · ^E a

换一组基，几何向量 a 没变，坐标 ^E a 会变。这就是坐标转换的根。

每个运算都要同时知道代数结果和几何意义
概念公式几何意义
范数||a||=√(aᵀa)向量长度
内积aᵀb=||a||||b||cosθ夹角、投影；为零表示正交
叉积a×b=a^∧b右手方向和有向面积；为 M07 的角速度公式做准备
矩阵乘法A(Bx)=(AB)x函数复合；右边的变换先作用
转置与逆RᵀR=I正交矩阵保持长度和夹角，逆等于转置
行列式det(R)=1合法旋转保持方向；-1 含反射
 
        [  0   -a_z   a_y ]
a^∧ =  [ a_z    0   -a_x ]
        [-a_y   a_x    0  ]

a × b = a^∧ b

先把点、方向和坐标系记号分开

  • ^A p：同一个几何点在坐标系 A 下的坐标。
  • ^A R_B：把 B 系中的方向坐标旋转到 A 系；三列分别是 B 的三根单位轴在 A 中的坐标。
  • ^A t_B：B 系原点在 A 系中的坐标。
  • ^A T_B：把 B 系坐标转换到 A 系的刚体变换，也就是“B 在 A 中的位姿”。
^A p = ^A R_B · ^B p + ^A t_B

点依赖原点，所以旋转后还要加平移；自由方向向量没有原点，只做旋转。速度究竟只旋转还是还要补偿坐标系运动，要看它是静态换基还是动态参考系问题。

从换基推导旋转矩阵

设 E_A、E_B 是两组正交基，同一个几何向量可以写成：

a = E_A · ^A a = E_B · ^B a

左乘 E_Aᵀ：
^A a = E_AᵀE_B · ^B a

所以：^A R_B = E_AᵀE_B

这直接解释了为什么 ^A R_B 的每一列就是 B 的一根轴在 A 中的坐标，也解释了 RᵀR=I、R⁻¹=Rᵀ。

二维车辆常用 x 向前、y 向左、逆时针为正。B 相对 A 逆时针旋转 θ：

           [ cosθ  -sinθ ]
^A R_B =  [ sinθ   cosθ ]

RᵀR = I        det(R) = 1        R⁻¹ = Rᵀ

主动还是被动：“把几何向量旋转 θ”和“保持向量不动、把坐标轴旋转 θ”会出现互为转置的矩阵。不要背正负号；先声明 ^A R_B 的方向，再用单位轴验证。

三维旋转有多种表示，但表示的仍是同一件事

旋转本身只有 3 个自由度，不同表示服务不同用途
表示形式优点主要边界
旋转矩阵R∈SO(3)直接变换点，组合清晰9 个数受正交与行列式约束
轴角/旋转向量φ=θu3 个参数，适合描述旋转增量角度周期与接近 π 时需谨慎
欧拉角roll / pitch / yaw人容易理解必须声明固定轴/随体轴和顺序，存在万向锁
单位四元数q=[w,v]、||q||=1紧凑、适合连续插值q 与 -q 同旋转；存储顺序不统一
 
q = [ cos(θ/2),  u sin(θ/2) ]

q⁻¹ = q* = [w, -v]      ||q|| = 1

[0,p′] = q ⊗ [0,p] ⊗ q⁻¹

nuScenes 四元数记录顺序为 w,x,y,z，ROS 消息字段通常排列为 x,y,z,w。插值或累乘前应归一化；连续序列中还要处理 q/-q 的符号连续性。

齐次坐标：把旋转和平移放进一次乘法

          [ ^A R_B   ^A t_B ]
^A T_B = [  0 0 0       1   ] ∈ SE(3)

[ ^A p ]           [ ^B p ]
[   1   ] = ^A T_B [   1   ]

齐次坐标最后一维区分对象类型：

点：     [p, 1]  → 旋转 + 平移
方向：   [v, 0]  → 只旋转，不平移

组合与求逆：用上下标检查相邻坐标系

传感器 S 的点先到车体 B，再到地图 M：

^M T_S = ^M T_B · ^B T_S

^M p = ^M T_B · ^B T_S · ^S p

中间的 B 像单位一样相消，右边的变换先作用。矩阵一般不可交换。若 ^A T_B=[R,t]，令变换后再变回来：

^A p = R · ^B p + t
^B p = Rᵀ(^A p - t) = Rᵀ · ^A p - Rᵀt

          [ Rᵀ   -Rᵀt ]
^B T_A = [ 0        1 ]

所以逆变换中的平移是 -Rᵀt，不是机械地写 -t。

两个数值例子：单级换系与两级坐标链

例 1：车辆点到地图

车辆原点在地图 (10,5)，yaw 为 90°；车辆系中一点 ^B p=(2,1)：

         [ 0  -1 ] [ 2 ]   [ 10 ]   [ 9 ]
^M p =  [ 1   0 ] [ 1 ] + [  5 ] = [ 7 ]

回代：^B p = Rᵀ(^M p - t) = (2,1)

例 2：传感器点经车体到地图

传感器与车体轴平行，传感器原点在车体 ^B t_S=(1,0.2)；传感器点 ^S p=(3,-0.2)：

^B p = I · (3,-0.2) + (1,0.2) = (4,0)

^M p = R(90°) · (4,0) + (10,5) = (10,9)

这两个例子应分别执行原点、单位轴和往返测试，而不是只看最终数字。

多传感器工程还要同时对齐时间

^G T_S(t) = ^G T_E(t) · ^E T_S

源传感器 S₁ 在 t₁ 的点变到目标传感器 S₂ 在 t₂：
^S₂ T_S₁ = [^G T_S₂(t₂)]⁻¹ · ^G T_S₁(t₁)

静态外参

传感器相对车体的安装关系近似固定，但仍需版本化标定与单位。

动态位姿

ego 相对 global 随时间变化；点和 pose 必须同一时刻或经过有定义的插值。

相机投影

外参只把点变到相机坐标；还需有效深度和内参才能得到像素。

经纬度

经纬度不是普通直角坐标，必须先声明基准、投影和局部原点。

为什么《视觉 SLAM 十四讲》下一步进入李群与李代数

合法旋转矩阵和位姿矩阵不是任意矩阵，它们分别组成 SO(3) 与 SE(3)。直接给矩阵元素加一个普通向量，可能破坏正交约束；因此常在局部切空间表达小增量，再通过指数映射回到群上。

R ∈ SO(3)        T ∈ SE(3)

φ ∈ R³           R = exp(φ^∧)
小转角：          R ≈ I + φ^∧

ξ = [ρ, φ] ∈ R⁶
      [ φ^∧  ρ ]
ξ^∧ = [  0    0 ]      T = exp(ξ^∧)

hat 把向量变成李代数矩阵，exp 把局部增量映射到位姿，log 做反向映射。注意有限变换中 ρ 不能无条件直接当作齐次矩阵里的平移 t。

本课边界：只需理解“群上存位姿、切空间表达小增量”。BCH、雅可比、伴随与优化中的左右扰动属于后续高级课程。

练习、检查与验收

  1. 用自己的话区分几何向量、坐标列向量和点。
  2. 从两根旋转后的单位轴推导二维 R(θ)，并证明 R⁻¹=Rᵀ。
  3. 独立推导 T⁻¹，解释平移为何是 -Rᵀt。
  4. 手算本页两个数值例子，并执行 T⁻¹(Tp) 往返测试。
  5. 解释 q 与 -q 为何表示同一个旋转，以及 wxyz/xyzw 混用会怎样。
  6. 画出 nuScenes 中 sensor→ego→global 的矩阵链，并说明两个不同时刻的传感器帧怎样连接。
  7. 用 1° 小转角比较精确旋转与 I+φ^∧，说明近似误差来自哪里。
