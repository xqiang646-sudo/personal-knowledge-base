阿克曼转向与运动学自行车模型推导

阿克曼几何回答“左右前轮分别应该转多少”，自行车模型回答“整车在这个转角下怎样运动”。它们通过同一个瞬时旋转中心、转弯半径和曲率连接起来。

贡献角色：导师原创推导　对象：规划与控制入门学习者　证据状态：平面低速运动学模型

统一约定：后轴中心为参考点；x 向前、y 向左、yaw 逆时针为正；δ>0 表示左转，v 为后轴中心有符号速度。

假设与变量瞬时旋转中心阿克曼几何自行车模型离散积分质心模型数值例子边界与误区资料

第一步：写出模型成立的前提

  • 车辆在平面运动，车身是刚体。
  • 车轮纯滚动，接地点速度沿车轮平面，横向速度为零。
  • 前轮转向、后轮不转向；忽略轮胎侧偏、悬架和载荷转移。
  • L 为轴距，W 为左右轮距，R 为后轴中心到 ICR 的有符号半径。
  • δ 是前轴中心虚拟轮的转角，不是方向盘角，也不是内外轮角的算术平均。

模型边界：这些假设适合低速、正常附着和几何推导。高速、低附着或激烈操纵时，必须考虑轮胎侧偏和车辆动力学。

第二步：纯滚动迫使车轮指向同一个 ICR

车轮不能侧滑，所以每个接地点的速度方向必须沿轮平面。平面刚体在某一瞬间等价于绕一个瞬时旋转中心 ICR 转动；每个车轮的速度都与“ICR 到车轮”的半径垂直。

车轮无横向速度
→
车轮平面确定速度方向
→
各轮法线相交
→
共同 ICR

后轮不转向，因此两后轮法线就是后轴方向，ICR 必须位于后轴延长线上。左转时，ICR 位于车辆左侧。

第三步：推导左右前轮的阿克曼关系

左转时，左前轮是内轮、右前轮是外轮。由两个直角三角形：

tan(δ_in)  = L / (R - W/2)
tan(δ_out) = L / (R + W/2)

因此：cot(δ_out) - cot(δ_in) = W / L

内轮离 ICR 更近，所以 |δ_in| > |δ_out|。若前轴中心的虚拟单轨转角为 δ：

tan(δ) = L / R

cot(δ) = [cot(δ_in) + cot(δ_out)] / 2

注意第二式平均的是 cot，不是直接平均两个转角。用有符号曲率 κ=1/R 可以同时覆盖左右转：

tan(δ_L) = Lκ / (1 - Wκ/2)
tan(δ_R) = Lκ / (1 + Wκ/2)

上式要求参考点、轮距和符号约定一致，并且目标曲率没有让分母接近零。真实转向连杆还会有标定误差和非理想 Ackermann 比例。

第四步：把左右轮合并为虚拟单轨

自行车模型把每一轴的左右车轮合并到轴中心，同时保留相同 ICR。后轴中心速度在车体系中为 [v,0]；前轴中心距后轴中心 [L,0]。

由平面刚体速度关系，前轴中心速度为：

v_front = v_rear + ψ̇ ẑ × [L,0]

        [ v  ]
      = [ Lψ̇ ]

前轮无侧滑，前轴速度必须与转角 δ 的轮平面平行。它与轮平面法向 [-sinδ, cosδ] 的点积为零：

[v, Lψ̇] · [-sinδ, cosδ] = 0

-v sinδ + Lψ̇ cosδ = 0

ψ̇ = (v/L) tanδ

转换到地图坐标，就得到以后轴中心为状态的运动学自行车模型：

ẋ = v cosψ
ẏ = v sinψ
ψ̇ = (v/L) tanδ

κ = dψ/ds = ψ̇/v = tanδ/L = 1/R

小角度近似：κ≈δ/L 只在 |δ| 很小时成立；一般情况必须使用 tanδ。

第五步：从微分方程得到离散轨迹

最简单的前向欧拉更新为：

x_(k+1) = x_k + v cosψ_k · Δt
y_(k+1) = y_k + v sinψ_k · Δt
ψ_(k+1) = ψ_k + (v/L)tanδ · Δt

当一个时间步内 v 与 δ 恒定，车辆实际走圆弧。令 ω=(v/L)tanδ、R=v/ω，可精确积分：

x_(k+1) = x_k + R[sin(ψ_k + ωΔt) - sinψ_k]
y_(k+1) = y_k - R[cos(ψ_k + ωΔt) - cosψ_k]
ψ_(k+1) = ψ_k + ωΔt

当 |δ| 接近零时，应切换到直线形式，避免计算无限大的 R。

参考点换到质心，方程也必须一起换

若状态点位于质心，质心到前后轴距离分别为 l_f、l_r，且 L=l_f+l_r。质心速度方向与车身朝向存在几何侧偏角：

β = atan[(l_r/L) tanδ]

ẋ = v cos(ψ + β)
ẏ = v sin(ψ + β)
ψ̇ = (v/l_r) sinβ = (v cosβ/L) tanδ

这里的 v 是质心速度大小。后轴模型、质心模型和动态自行车模型不能只挑几条公式混在一起；首先要固定参考点和速度定义。

数值例子：从虚拟转角得到真实轮角和轨迹

取 L=2.8 m、W=1.6 m、δ=10°、v=5 m/s：

R = L/tanδ = 15.8796 m
κ = 1/R = 0.062974 m⁻¹

δ_in  = 10.519°
δ_out =  9.529°

ψ̇ = vκ = 0.31487 rad/s ≈ 18.041°/s

若从 (x,y,ψ)=(0,0,0) 开始保持控制 0.1 s，精确圆弧积分得到约：

Δψ = 1.804°
x = 0.4999 m
y = 0.00787 m

这个例子同时说明：虚拟前轮转角位于内外轮角之间，但不是它们的精确算术平均。

模型边界与常见误区

方向盘角当车轮角

真实车辆还有转向传动比、零偏、执行器延迟和连杆非线性。

混淆 L 与 W

轴距决定基本曲率关系，轮距决定内外轮转角差。

最小半径没有参考点

后轴中心、质心、外轮和车身外角的轨迹半径不同。

倒车符号错误

同一 δ 下，v<0 会使 ψ̇ 反号；必须统一路径方向和速度符号。

欧拉步长过大

高曲率或长步长会让直线近似偏离圆弧，应减小步长或使用精确积分。

高速仍假设无侧滑

动力学模型还要包含横向速度、yaw rate、轮胎侧偏角、轮胎力、质量和转动惯量。
