阿克曼转向与运动学自行车模型完整推导

阿克曼几何回答“左右前轮分别应该转多少”，自行车模型回答“整车在虚拟转角下怎样运动”。本课不从最终公式开始，而是从叉积、刚体点速度和车轮无侧滑约束一步步推出它们。

贡献角色：导师原创推导　对象：规划与控制入门学习者　证据状态：平面低速运动学模型

统一约定：后轴中心为参考点；x 向前、y 向左、yaw 逆时针为正；δ>0 表示左转，v 为后轴中心沿车身 x 轴的有符号速度。

第一步：写出模型成立的前提

  • 车辆在平面运动，车身是刚体。
  • 车轮纯滚动，接地点速度沿车轮平面，横向速度为零。
  • 前轮转向、后轮不转向；忽略轮胎侧偏、悬架和载荷转移。
  • L 为轴距，W 为左右轮距，R 为后轴中心到 ICR 的有符号半径。
  • δ 是前轴中心虚拟轮的转角，不是方向盘角，也不是内外轮角的算术平均。

模型边界：这些假设适合低速、正常附着和几何推导。高速、低附着或激烈操纵时，需要轮胎侧偏、横向力、质量和转动惯量等动力学量。

第二步：从小角度位移得到刚体点速度

设车体在短时间 Δt 内逆时针转过 Δψ。相对参考点 O 的位置向量为 r，由小角度几何：

Δr ≈ Δψ · ẑ × r

两边除以 Δt，并令 Δt→0：

ṙ = ψ̇ ẑ × r = ω × r

对平面位置 r=[r_x,r_y,0]ᵀ、角速度 ω=[0,0,ψ̇]ᵀ：

       [ -ψ̇ r_y ]
ω×r = [  ψ̇ r_x ]
       [    0     ]

左转时 ψ̇>0，位于车体前方的点 r_x>0 会获得向左的附加速度，符合直觉。

若 O 本身还有平移速度，刚体上任意点 P 的总速度为：

v_P = v_O + ω × r_OP

第三步：轮胎无侧滑约束与瞬时旋转中心

转角为 α 的车轮，其滚动方向和横向法向分别是：

e(α) = [ cosα,  sinα ]ᵀ
n(α) = [-sinα,  cosα ]ᵀ

纯滚动横向约束：n(α)ᵀ v_wheel = 0

即车轮速度只能沿轮平面。平面刚体在某一瞬间可看作绕 ICR 转动；ICR 的瞬时速度为零。若参考点速度为 [v_x,v_y]：

0 = [v_x, v_y]ᵀ + [-ψ̇ r_y, ψ̇ r_x]ᵀ

r_x = -v_y/ψ̇        r_y = v_x/ψ̇

以后轴中心为参考点，v_r=[v,0]ᵀ，所以 ICR 位于 [0,v/ψ̇]ᵀ，后轴中心半径：

R = v/ψ̇

ψ̇ = 0 时，ICR 在无穷远，车辆直行。

第四步：由刚体速度直接推导阿克曼关系

左、右前轮相对后轴中心的位置分别是 r_L=[L,W/2]ᵀ、r_R=[L,-W/2]ᵀ。代入刚体点速度：

      [ v - ψ̇W/2 ]            [ v + ψ̇W/2 ]
v_L = [    Lψ̇    ]      v_R = [    Lψ̇    ]

前轮无侧滑，因此各自轮平面必须和速度方向平行：

tan(δ_L) = Lψ̇ / (v - ψ̇W/2)
tan(δ_R) = Lψ̇ / (v + ψ̇W/2)

车辆运动且 v≠0 时，令转向曲率 κ_steer=ψ̇/v，得到左右转都适用的形式：

tan(δ_L) = Lκ_steer / (1 - Wκ_steer/2)
tan(δ_R) = Lκ_steer / (1 + Wκ_steer/2)

左转时 κ_steer>0，左前轮是内轮，角度更大。车辆静止时虽然不能用 ψ̇/v 做比值，仍可由转向几何定义 κ_steer=tanδ/L。用 R=1/κ_steer 也可写成传统直角三角形形式：

tan(δ_in)  = L / (R - W/2)
tan(δ_out) = L / (R + W/2)

cot(δ_out) - cot(δ_in) = W/L

第五步：把左右轮合并为虚拟单轨

自行车模型把每一轴的左右车轮合并到轴中心，并保留相同 ICR。前轴中心相对后轴中心是 r_f=[L,0]ᵀ：

v_front = [v,0]ᵀ + ψ̇ ẑ × [L,0]ᵀ
        = [v, Lψ̇]ᵀ

[-sinδ, cosδ] · [v, Lψ̇] = 0

-v sinδ + Lψ̇ cosδ = 0

ψ̇ = (v/L) tanδ

转换到地图坐标，得到以后轴中心为状态的运动学自行车模型：

ẋ = v cosψ
ẏ = v sinψ
ψ̇ = (v/L) tanδ

虚拟前轮转角满足 tanδ=L/R，并与两个真实轮角满足：

cotδ = [cot(δ_in) + cot(δ_out)] / 2

平均的是 cot，不是两个真实轮角的算术平均。自行车模型抽象掉轮距 W；底层转向机构再用 Ackermann 几何和真实连杆标定恢复左右轮角。

曲率与倒车：区分弧长 s 和有符号行程 σ

这里必须区分两个看起来相似的量：

几何弧长：      ds/dt = |v|      s 始终不减
有符号行程：    dσ/dt = v        前进增加，倒车减少

转向曲率：κ_steer = dψ/dσ = ψ̇/v = tanδ/L

当前进且路径点序与车辆运动方向一致时，s=σ，转向曲率与路径有符号曲率相同。倒车时：

κ_path = dχ/ds = ψ̇/|v|

v<0 时，同一 δ 会使 ψ̇ 反号，
因此按时间前进观察的轨迹曲率与 κ_steer 符号相反。

不要混写：ψ̇/v 对应有符号行程，不是非负累计弧长。规划路径的点序、车辆朝向和前进/倒车挡位必须共同决定曲率符号。

第六步：从微分方程积分出离散轨迹

导数描述瞬时变化率。把一个短时间步内的变化率乘以 Δt，得到最简单的前向欧拉更新：

x_(k+1) = x_k + v cosψ_k · Δt
y_(k+1) = y_k + v sinψ_k · Δt
ψ_(k+1) = ψ_k + (v/L)tanδ · Δt

若一个时间步内 v 与 δ 恒定，车辆实际走圆弧。令 ω=(v/L)tanδ、R=v/ω=L/tanδ：

x_(k+1) = x_k + R[sin(ψ_k + ωΔt) - sinψ_k]
y_(k+1) = y_k - R[cos(ψ_k + ωΔt) - cosψ_k]
ψ_(k+1) = ψ_k + ωΔt

当 |δ| 接近零时，应使用直线极限或稳定的 sinc 形式，避免显式计算无限大的 R。

参考点换到质心，方程也必须一起换

若状态点位于质心，质心到前后轴距离分别为 l_f、l_r，且 L=l_f+l_r。运动学几何使质心速度方向与车身朝向存在角度：

β = atan[(l_r/L) tanδ]

ẋ = v cos(ψ + β)
ẏ = v sin(ψ + β)
ψ̇ = (v/l_r) sinβ = (v cosβ/L) tanδ

这里的 v 是质心速度大小，β 是运动学模型中的几何速度方向差，不等同于由轮胎力产生的完整动力学侧偏状态。后轴模型、质心模型和动态自行车模型不能拼接使用。

数值例子：从虚拟转角得到真实轮角和轨迹

取 L=2.8 m、W=1.6 m、δ=10°、v=5 m/s：

R = L/tanδ = 15.8796 m
κ_steer = 1/R = 0.062974 m⁻¹

δ_in  = 10.519°
δ_out =  9.529°

ψ̇ = vκ_steer = 0.31487 rad/s ≈ 18.041°/s

若从 (x,y,ψ)=(0,0,0) 开始保持控制 0.1 s，精确圆弧积分得到约：

Δψ = 1.804°
x = 0.4999 m
y = 0.00787 m

若只用小角度近似 κ≈δ/L，δ=30° 时会比精确 tanδ/L 低估约 9.3%，所以它只能用于足够小的转角。

练习、边界测试与验收

  1. ψ̇=0.2 rad/s、r=(3,1) m，计算旋转附加速度 (-0.2,0.6) m/s，并解释符号。
  2. 若参考点速度为 (5,0) m/s，求该点总速度 (4.8,0.6) m/s。
  3. v=5 m/s、ψ̇=0.25 rad/s，求 ICR 并说明它在车体左侧 20 m。
  4. 从轮胎法向量与点积为零，独立推出 ψ̇=(v/L)tanδ。
  5. 重新计算本页 L/W/δ 例题，并验证左转时内轮角大于外轮角。
  6. 令 v=-5 m/s，分别说明 yaw rate、转向曲率与按时间前进的轨迹曲率怎样变号。
  7. 为离散模型设计 δ=0、v=0、κ→-κ 三组边界测试。
  8. 给出一个运动学模型失效场景，并列出动力学模型至少需要增加的状态或参数。
