曲率的定义与工程计算

曲率描述“路径方向随前进距离改变得多快”。它是连接路径几何、车辆转角、横摆角速度和横向加速度的核心量。

贡献角色：导师原创推导　对象：规划、控制与轨迹处理入门学习者　证据状态：二维路径几何

统一约定：右手 x-y 平面，路径按点序前进，逆时针/左转为正；曲率单位为 m⁻¹。

定义连续曲线离散三点航向差分方法对比车辆关联数值稳定练习资料

第一性定义：单位弧长带来多少方向变化

令 s 为沿路径累计的弧长，ψ(s) 为路径切线航向角，则有符号曲率定义为：

κ = dψ/ds

转弯半径：R = 1/|κ|
直线：κ = 0，R → ∞

左转时航向随弧长增加，κ>0；右转时 κ<0。同一条几何路径如果把点序反过来，有符号曲率会反号。

量纲检查：航向角用弧度、弧长用米，因此曲率单位是 rad/m，弧度无量纲，通常写成 m⁻¹。

连续参数曲线的曲率

对二维正则曲线 r(t)=(x(t),y(t))，速度向量不能为零。由切线方向对弧长求导可得：

            x′(t)y″(t) - y′(t)x″(t)
κ(t) = --------------------------------
        [x′(t)² + y′(t)²]^(3/2)

分子是速度与加速度的二维叉积，决定转向方向；分母消除参数走得快慢的影响，所以同一条曲线换一种参数化，曲率不变。

若路径直接写成 y=f(x)：

              f″(x)
κ(x) = -------------------
        [1 + f′(x)²]^(3/2)

圆 r(θ)=(R cosθ,R sinθ) 代入后得到 κ=1/R，与定义一致。

离散路径：用三点外接圆估计曲率

给定按路径顺序排列的三点 p0,p1,p2：

a = |p1 - p0|      b = |p2 - p1|      c = |p2 - p0|
cross = (p1 - p0)_x(p2 - p0)_y - (p1 - p0)_y(p2 - p0)_x

κ = 2·cross / (a·b·c)
  = 4·A_signed / (a·b·c)

cross=2A_signed。把 cross 当成三角形面积，会产生最常见的两倍误差。三点共线时 cross 为零，曲率为零；有重复点或边长接近零时，结果无定义，应拒绝该样本。

数值例子

取 p0=(0,0)、p1=(1,0)、p2=(1,1)：

a=1，b=1，c=√2，cross=1

κ = 2/(√2) = √2 ≈ 1.414 m⁻¹
R = 1/κ ≈ 0.707 m
function curvature3Points(p0, p1, p2, eps):
    a = norm(p1 - p0)
    b = norm(p2 - p1)
    c = norm(p2 - p0)
    if min(a, b, c) < eps:
        return INVALID
    cross = cross2d(p1 - p0, p2 - p0)
    return 2 * cross / (a * b * c)

已有航向角时：对弧长做差分

如果 ψ_i 与 s_i 是同一批路径点上的航向和累计弧长，可用中心差分：

κ_i ≈ wrapToPi(ψ_(i+1) - ψ_(i-1))
      ---------------------------------
             s_(i+1) - s_(i-1)

如果航向来自点 p_i 前后的两条线段，它们实际位于线段中点，应使用两中点之间的距离：

ψ_- = atan2((p_i-p_(i-1))_y, (p_i-p_(i-1))_x)
ψ_+ = atan2((p_(i+1)-p_i)_y, (p_(i+1)-p_i)_x)

κ_i ≈ wrapToPi(ψ_+ - ψ_-)
      ----------------------
          (ℓ_- + ℓ_+)/2

必须处理角度跳变：从 179° 到 -179° 实际只转了 2°。差分前要 unwrap 航向序列，或对每个角度增量执行 wrapToPi。

三种方法怎样选择

输入形式决定公式，数据质量决定稳定性
方法输入优点主要风险
连续导数解析曲线或可微样条定义直接，可在任意位置计算二阶导数放大拟合与噪声误差
三点外接圆有序离散点不需要显式航向，实现简单重复点、短基线和噪声会导致抖动
航向/弧长差分航向与弧长物理意义直观，便于连接 yaw rate角度 wrap、采样位置和非等距问题

曲率怎样连接车辆转角和动态约束

以后轴中心为参考点的运动学自行车模型：

κ = tanδ/L        δ = atan(Lκ)

ψ̇ = vκ

κ_max = tan(δ_max)/L
R_min = 1/|κ_max|

横向加速度近似：a_y = v²κ

曲率是几何量，但相同曲率在更高速度下产生更大的横向加速度。规划不能只检查路径“转不转得过来”，还要结合速度、舒适性和轮胎附着。

侧滑时要换角度：路径切线角是 χ=ψ+β。高速或明显侧偏时，路径曲率是 χ̇/v，不能无条件把车身 yaw rate ψ̇ 除以车速当成真实路径曲率。

工程数据中的数值稳定性

重复点与短基线

先去重并设最小弧长阈值；极近点会把毫米级噪声放大为巨大曲率。

非均匀采样

不要套等距差分。优先按弧长重采样，并记录实际采样间隔。

噪声的二阶放大

可用局部多项式、样条或圆拟合，但必须记录窗口，避免跨过真实尖角。

尖角无有限曲率

折线顶点的切线不连续，曲率本身不定义；平滑是在改变路径，不是恢复隐藏真值。

近直线半径爆大

优先报告接近零的 κ，不要直接展示数值巨大的 R；为零附近设置语义阈值。

大坐标消减精度

先减局部原点再计算差分。旋转平移保持曲率，反射会翻转符号，缩放会改变大小。

图像坐标的 y 轴通常向下，相当于对常用车辆平面做了反射；直接在像素坐标算有符号曲率时，正负号可能与车辆坐标相反。

实习生练习

  1. 从 κ=dψ/ds 推导半径为 R 的圆为什么有 |κ|=1/R。
  2. 手算三点 (0,0),(1,0),(1,1) 的曲率，并解释正号来自哪里。
  3. 构造一组跨越 ±π 的航向角，比较直接相减和 wrap 后的结果。
  4. 给定 L=2.8 m、δ_max=30°，计算后轴中心的 κ_max 与 R_min。
  5. 对同一条圆弧分别稀疏采样和密集采样，加入相同位置噪声，观察三点曲率抖动。
