从微积分和线性代数推导曲率

曲率描述“路径方向随前进距离改变得多快”。本课先补向量、行列式、导数、参数曲线、链式法则和弧长，再推导连续公式、离散三点公式与工程计算方法。

贡献角色：导师原创推导　对象：规划、控制与轨迹数据处理入门学习者　证据状态：二维有向路径几何

统一约定：右手 x-y 平面，路径按点序前进，逆时针/左转为正；s 是非负累计弧长，曲率单位为 m⁻¹。

数学先修：两条依赖链

线性代数链

点差得到位移向量 → 范数给长度 → 内积给夹角 → 二维行列式给有向面积与左右转 → 归一化得到单位切向量。

微积分链

极限 → 导数是瞬时变化率 → 向量函数逐分量求导 → 链式法则 → 弧长参数 → 对单位距离求方向变化。

先把每个符号翻译成几何意义
概念公式本课作用
位移向量Δp=p₁-p₀相邻路径点的方向与距离
范数||v||=√(v·v)线段长度和参数曲线的“走得多快”
内积a·b=||a||||b||cosθ夹角、投影与正交
二维行列式det(a,b)=a_xb_y-a_yb_x有向面积和从 a 到 b 的左右转
向量导数r′(t)=(x′(t),y′(t))曲线的参数切向导数
链式法则dψ/ds=(dψ/dt)/(ds/dt)消除参数走得快慢的影响
 
u=(3,4)      v=(-4,3)

||u||=||v||=5
u·v=0                 两向量正交
det(u,v)=25 > 0       从 u 到 v 逆时针转

r′(t) 一般称为参数切向导数；只有参数 t 确实是物理时间时，它才是物理速度。

第一性定义：单位弧长带来多少方向变化

设正则参数曲线 r(t)=(x(t),y(t)) 且 r′(t)≠0。从起点到参数 t 的弧长：

          t
s(t) = ∫ ||r′(u)|| du
          t₀

ds/dt = ||r′(t)||

把切向导数归一化得到单位切向量 T，并用 ψ 表示其航向：

T = r′/||r′|| = (cosψ, sinψ)

κ = dψ/ds

转弯半径：R = 1/|κ|      直线：κ=0，R→∞

左转时航向随点序增加，κ>0；右转时 κ<0。弧度无量纲，弧长用米，所以曲率通常写成 m⁻¹。

从单位切向量逐步推导一般参数公式

为减少符号，令 v=r′(t)、q=||v||=ds/dt。这里的 v 是参数切向导数，不一定是物理速度。

T = v/q = (cosψ, sinψ)

dT/dt = (dψ/dt)(-sinψ, cosψ)

det(T, dT/dt) = dψ/dt

另一方面，对 T=v/q 使用商法则：

dT/dt = v′/q - vq′/q²

det(T, dT/dt)
= det(v/q, v′/q - vq′/q²)
= det(v,v′)/q²

因为 det(v,v)=0，所以包含 q′ 的项消失。

两种表达相等，再用链式法则除以 ds/dt=q：

dψ/dt = det(r′,r″) / ||r′||²

κ = dψ/ds
  = (dψ/dt)/(ds/dt)
  = det(r′,r″) / ||r′||³

            x′(t)y″(t) - y′(t)x″(t)
κ(t) = --------------------------------
        [x′(t)² + y′(t)²]^(3/2)

分子决定转向方向，分母消除正向参数化的快慢。若重参数 t=g(u)，分子乘 g′³、分母乘 |g′|³：g′>0 时有符号曲率不变，g′<0 反向遍历时符号翻转。

用函数曲线和圆验证公式

函数曲线 y=f(x)

令参数 t=x，则 r′=(1,f′)、r″=(0,f″)：

              f″(x)
κ(x) = -------------------
        [1 + f′(x)²]^(3/2)

半径为 R 的逆时针圆

令 r(t)=(R cos t,R sin t)，代入一般公式：

det(r′,r″)=R²      ||r′||³=R³

κ = R²/R³ = 1/R

把圆按顺时针方向参数化，几何圆不变，但有向点序相反，因此 κ=-1/R。

离散三点公式：从外接圆和有向面积推导

给定按路径顺序排列的三个不同点 p₀,p₁,p₂。若不共线，它们确定唯一外接圆。定义：

a = ||p₁-p₀||      b = ||p₂-p₁||      c = ||p₂-p₀||

D = det(p₁-p₀, p₂-p₀)
A_signed = D/2

令 γ 为向量 p₁-p₀ 与 p₂-p₀ 的夹角，则：

|A| = (1/2)ac sinγ
b = 2R sinγ

消去 sinγ：|A| = abc/(4R)

恢复方向符号：
κ = 4A_signed/(abc)
  = 2D/(abc)

这是三点的 Menger 曲率。三点在同一圆上时结果精确；对一般光滑曲线，只有采样窗口收缩时才趋近局部曲率。宽窗口、强不等距采样或尖角处，不能把它直接称为 p₁ 的真实曲率。

手算例子

p₀=(0,0)、p₁=(1,0)、p₂=(1,1)：

a=1，b=1，c=√2，D=1

κ = 2/√2 = √2 ≈ 1.414 m⁻¹
R ≈ 0.707 m

重复点使分母为零，应返回 INVALID；三个不同但共线的点有 D=0，可解释为离散曲率零。

已有航向角时：对弧长做差分

若 ψ_i 与 s_i 是同一批路径点的切向航向和累计弧长：

κ_i ≈ wrapToPi(ψ_(i+1) - ψ_(i-1))
      ---------------------------------
             s_(i+1) - s_(i-1)

这个差商首先表示区间上的平均曲率；只有窗口足够小且曲率平滑时，才近似赋给中间点。

若航向来自点 p_i 前后的两条线段，它们位于线段中点，应使用两个中点之间的距离：

ψ_- = atan2((p_i-p_(i-1))_y, (p_i-p_(i-1))_x)
ψ_+ = atan2((p_(i+1)-p_i)_y, (p_(i+1)-p_i)_x)

κ_i ≈ wrapToPi(ψ_+ - ψ_-)
      ----------------------
          (ℓ_- + ℓ_+)/2

必须处理角度跳变：179°→-179° 实际只转了 +2°，直接相减却得到 -358°。差分前要 unwrap 航向序列，或对每个角度增量执行 wrapToPi。

实验 3A：角度 wrap

179°→-179°、Δs=1 m：直接差分约 -6.248 m⁻¹，wrap 后约 +0.0349 m⁻¹。

实验 3B：大坐标 float32

把 1 m 间距点整体平移到 10⁸ 附近再降为 float32，差值可能坍缩。应先在高精度中减局部原点，再降精度。

实验报告应固定真值、随机种子、噪声强度和窗口，报告 bias、RMSE、P95 与无效点比例，不能只挑一张“看起来平滑”的图。

三种计算方法怎样选择

输入形式决定公式，采样和噪声决定稳定性
方法输入优点主要风险
连续导数解析曲线或可微样条定义直接，可在任意位置计算二阶导数放大拟合与噪声误差
三点外接圆有序离散点不需要显式航向，实现简单窗口、重复点、短基线与噪声
航向/弧长差分航向与弧长物理意义直观，便于连接 yaw rate角度 wrap、采样位置和平均窗口

曲率怎样连接车辆转角和动态约束

M07 中以后轴中心和有符号行程 σ 定义转向曲率：

κ_steer = dψ/dσ = tanδ/L
δ = atan(Lκ_steer)

若车辆前进且轨迹切向 χ=ψ：
κ_path = dχ/ds = ψ̇/|v| = κ_steer

κ_steer,max = tan(δ_max)/L
横向加速度近似：a_y = speed² · κ_path

倒车时 dσ/dt=v<0，而 ds/dt=|v|。此时转向曲率与按时间前进观察的路径曲率不能直接混成一个符号。

侧滑时还要换角度：路径切线角是 χ=ψ+β。高速或明显侧偏时，路径曲率应由 χ 变化计算，不能无条件把车身 yaw rate 除以速度当作真实路径曲率。

工程数据中的数值稳定性清单

重复点与短基线

先去重并设最小弧长阈值；极近点会放大毫米级噪声。

非均匀采样

不要套等距差分。按弧长重采样，或使用保留真实间距的公式。

二阶导数放大噪声

可用局部多项式、样条或圆拟合，但必须记录窗口和边界策略。

尖角没有有限曲率

折线顶点切线不连续；平滑是在改变路径，不是恢复隐藏真值。

近直线半径爆大

优先报告接近零的 κ，而不是数值巨大的 R；为零附近设置语义阈值。

坐标变换与符号

旋转平移保持曲率；反射或逆序翻转有符号曲率；缩放会改变曲率大小。

图像坐标 y 轴通常向下，相当于对常用车辆平面做反射；直接在像素坐标中计算时，曲率符号可能与车辆坐标相反。
